canflex2Transceiver (S32K3xx)
=============================

Overview
--------

The module ``canflex2Transceiver`` implements the ``ICanTransceiver``
interface of ``cpp2can`` on top of the ``bspFlexCan`` device driver,
providing CAN frame transmission and reception to the generic communication
stack.
