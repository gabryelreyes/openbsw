bspMcu (S32K3xx)
================

Overview
--------

The module ``bspMcu`` provides the S32K344 device register headers (NXP
peripheral access layer, vendored under ``include/3rdparty/nxp``), the CMSIS
Cortex-M7 core integration via ``mcu/mcu.h`` and the software system reset.
